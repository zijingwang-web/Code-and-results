Each figure in this study was generated from its corresponding statistical or network result file. To facilitate reproducibility and data verification, the relationships between the figure names used in the paper and their associated data files are summarized below：**fig02** was generated from `overall\_color\_ratio\_histogram\_linear.csv`. This file records the overall distribution of the number of aggregated colors used on each book cover, including the number of colors, the corresponding book count, the percentage, and the total number of books. It is used to show the overall proportion of color-count usage across all book covers.**fig03** was generated from `per\_category\_color\_count\_histograms\_3x11\_percentage.csv`. This file records, for each category, the number and percentage of books associated with different numbers of aggregated colors. It is used to show differences in color-count distributions across the 33 categories.**fig04** was generated from `category\_color\_ratios\_data\_with\_global.csv`. This file records the aggregated color-ratio data for each category as well as the global baseline. It is used to show differences in color composition across categories, with the overall color distribution provided as a reference.**fig05** was generated jointly from `network\_nodes.csv` and `network\_edges\_weight.csv`. The file `network\_nodes.csv` provides information on color nodes, including community assignments, node positions, and node attributes, while `network\_edges\_weight.csv` provides the connections between color nodes and their edge weights. Together, these files are used to visualize the color co-occurrence network and its community structure.**fig06** was generated with support from `color\_pair\_frequency.txt` and `node\_pair\_frequency\_log\_fit\_info.txt`. The file `color\_pair\_frequency.txt` records the co-occurrence frequency, color-pair information, and ranking of color combinations, while `node\_pair\_frequency\_log\_fit\_info.txt` records the logarithmic fitting results of the color-pair frequency distribution. These files are used to show the frequency distribution of color-pair usage and its rank-decay pattern.**fig07** was generated from `simulated\_pair\_probs\_tau0.550.csv`. This file records the simulated counts, probabilities, and rankings of color pairs. It is used to show the color-pair probability distribution obtained from the Gibbs selection model and the semi-log trend of the Top-N simulated color combinations.
