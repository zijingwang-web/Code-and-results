# Code-and-results

Code and Results for the Manuscript: *Color Usage Patterns in Book Cover Design from a Complex-Network Perspective*

## Overview

This repository contains the supplementary code and result data associated with the manuscript *Color Usage Patterns in Book Cover Design from a Complex-Network Perspective*. The materials are organized to support reproducibility, verification, and further analysis of the figures and results reported in the paper.

## Repository Structure

```text
Code-and-results/
├── Supplementary_Code/
│   ├── requirements
│   ├── code files
│   └── README files
├── results/
│   ├── Data_files/
│   │   ├── figure-related data files
│   └── README file
└── README.md
```

## Supplementary_Code

The `Supplementary_Code` folder contains the core code used in the manuscript. It includes multiple code files corresponding to different stages of the analysis, such as color extraction, color co-occurrence network construction, network metric calculation, community detection, statistical visualization, and color-pair simulation.

This folder also contains README files that explain the purpose, input requirements, output files, and usage of the corresponding code modules.

## results

The `results` folder contains the data files corresponding to the figures presented in the manuscript. These files provide the underlying statistical results, network data, and simulation outputs used to generate the final figures.

A detailed README file is also included in the `results` folder to explain the relationship between each figure in the manuscript and its corresponding data file.

## Notes

The code and result files are provided to facilitate transparency and reproducibility. Users may refer to the README files inside each folder for more detailed explanations of the individual scripts, data files, and figure-generation outputs.
